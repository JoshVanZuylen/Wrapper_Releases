# Wrapper

Professional decal placement and trim sheet workflows for 3ds Max.

Wrapper gives 3D artists a complete toolkit for applying trim sheets, placing mesh decals, and generating strip details from a single, themeable interface inside 3ds Max.

- 3ds Max 2024
- 3ds Max 2025
- 3ds Max 2026
- 3ds Max 2027

## Quick Install

1. Download and extract the release zip that matches your Max version.
2. Open 3ds Max and drag `Wrapper_MacroInstaller.ms` into the viewport.
3. The tool launches automatically on first install.

See the [Installation](https://github.com/JoshVanZuylen/Wrapper_Releases/wiki/Installation) guide for full details.
